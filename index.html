Battery Bar Card

Een custom Lovelace card voor Home Assistant die batterijniveaus weergeeft als een horizontale gesegmenteerde balk met een groot percentage getal er overheen.

A custom Home Assistant Lovelace card that displays battery levels as a horizontal segmented bar with a large percentage number overlaid.

🔧 Installatie

Via HACS (aanbevolen)

  1. Ga in Home Assistant naar HACS → Frontend
  2. Klik op de drie puntjes rechtsboven → Custom repositories
  3. Voer de repository URL in en kies categorie Lovelace
  4. Klik op Add, zoek daarna naar Battery Bar Card en installeer
  5. Herstart Home Assistant of clear de browsercache

Handmatig

  1. Download battery-bar-card.js
  2. Zet het bestand in /config/www/
  3. Voeg toe aan je Lovelace resources:
resources:
  - url: /local/battery-bar-card.js
    type: module

⚙️ Configuratie

Minimaal voorbeeld

type: custom:battery-bar-card
entities:
  - sensor.logitech_mous_battery_level
  - sensor.pixel_7_battery_level

Volledig voorbeeld

type: custom:battery-bar-card
title: Batterijen
entities:
  - sensor.logitech_mous_battery_level
  - sensor.pixel_7_battery_level
  - sensor.s23_ultra_battery_level
height: 15
font_size: 30
font_color: "#ffffff"
font_weight: 800
segments: 10
low_threshold: 15
mid_threshold: 30
show_name: false
decimals: 1

🔧 Installation

Via HACS (recommended)

  1. In Home Assistant, go to HACS → Frontend
  2. Click the three dots top right → Custom repositories
  3. Enter the repository URL and choose category Lovelace
  4. Click Add, then search for Battery Bar Card and install
  5. Restart Home Assistant or clear your browser cache

Manual

  1. Download battery-bar-card.js
  2. Place the file in /config/www/
  3. Add to your Lovelace resources:
resources:
  - url: /local/battery-bar-card.js
    type: module

⚙️ Configuration

Minimal example

type: custom:battery-bar-card
entities:
  - sensor.logitech_mous_battery_level
  - sensor.pixel_7_battery_level

Full example

type: custom:battery-bar-card
title: Batteries
entities:
  - sensor.logitech_mous_battery_level
  - sensor.pixel_7_battery_level
  - sensor.s23_ultra_battery_level
height: 15
font_size: 30
font_color: "#ffffff"
font_weight: 800
segments: 10
low_threshold: 15
mid_threshold: 30
show_name: false
decimals: 1

🔄 Compatibility

  • Home Assistant 2023.0 or newer
  • Works with all sensors that return a value of 0–100
  • Also supports the battery_level attribute
